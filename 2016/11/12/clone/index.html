二.浅赋值 

//基本数据类型 String，Number,Boolean,null,underfined 按值传递
    复合数据类型 Array,Object 按引用传递 
    如果将变量类比成一个盒子，那么基本数据类型的变量装的就是实在的内容，通过"="号赋值，就是复制多一份出来了

    复合数据类型的变量装的一把钥匙，可以通过这个钥匙找到我们所要的数据，通过=赋值，就是复制多了一把钥匙，数据还是只要一份

        name:"shafee",
        age:20,
        city:"广州",
        tel:[13800138000,10086],
        say:function(){
            console.log(this.age)
        }
    }


var p1=person;

    }
}
